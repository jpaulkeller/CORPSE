Names: Elves, Female
source or inspiration: James Hutchings, original information from J.R.R. Tolkien.
Thanks to Wikipedia for the lists that these tables are based on.
 
This table and the one below freely mix the two main elven languages in Tolkien's books (Quenya and Sindarin), as well as the various different kinds of names that elves could have (such as seperate names given by their mother and father)
 
Roll 3 dice.


1st dice 2nd dice 3rd dice result
Roll again but add 'Curu' to the start of the name. For example 'Daeron' becomes 'Curudaeron'. Means 'Skilled'.
Roll again but add 'Pitya' to the start of the name. Means 'Little'.
Roll again but add 'Mori' to the start of the name. Means 'Dark'.
Roll again but add 'Telun' to the start of the name. Means 'Last' (implying 'youngest descendent of...').
Roll again but add 'Turka' to the start of the name. Means 'Strong-willed'.
Roll again but get rid of the last syllable and replace it with 'rod'. For example 'Gelmir' becomes 'Gelrod'. Means 'Mighty descendent of...'.
Roll again but if the name ends in a consonant, add 'ion' at the end (for example 'Gelmir' becomes 'Gelmirion'). Or if the name ends in a vowel, get rid of that letter and add 'ion' at the end (for example 'Ingwe' becomes 'Ingwion'). Means 'Son of..'.
